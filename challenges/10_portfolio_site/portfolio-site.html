Challenge Question:

Build a fully responsive personal portfolio page with light/dark theme and adaptable sections. Ensure the page includes a well-designed back button (using an <a> tag linking to index.html) and displays the challenge question prominently on the page. Add all necessary meta tags for SEO, responsiveness, and include a favicon. Make the page feel modern, interactive and engaging using only HTML and CSS. Use media queries to make responsive. It should be mobile friendly.

  • Concepts Covered:
  • Container Queries
  • CSS Nesting
  • CSS Variables
  • Responsive layout
  • accent-color for form inputs
  • :has() selector (if supported)
  • Accessibility: color contrast, :focus-visible, semantic tags
  • Bonus: Use BEM or SMACSS naming convention.
