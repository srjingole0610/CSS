CSS SMACSS Methodology

Scalable and Modular Architecture for CSS: Organize, scale, and maintain your stylesheets with confidence!

What is SMACSS?

SMACSS (Scalable and Modular Architecture for CSS) is a flexible CSS methodology created by Jonathan Snook. It provides a set of guidelines for categorizing CSS rules, making your stylesheets more scalable, maintainable, and modular as your project grows.

  • Base Rules: Default element styles (e.g., body, h1, ul).
  • Layout Rules: Major page sections and grid structures (e.g., .layout-header, .layout-sidebar).
  • Module Rules: Reusable, self-contained components (e.g., .card, .nav).
  • State Rules: Styles for different states or variations (e.g., .is-active, .is-hidden).
  • Theme Rules: Visual design variations (e.g., .theme-dark, .theme-light).

SMACSS Rule Types

/* Base */
body { font-family: sans-serif; }
h1 { font-size: 2rem; }

/* Layout */
.layout-header { ... }
.layout-sidebar { ... }

/* Module */
.card { ... }
.card__title { ... }

/* State */

.is-hidden { ... }

/* Theme */

.theme-light { ... }
