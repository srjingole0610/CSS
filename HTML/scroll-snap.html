CSS Scroll Snap

Create smooth, precise, and user-friendly scroll experiences with just CSS!

What is CSS Scroll Snap?

CSS Scroll Snap is a modern CSS feature that lets you create scrollable containers where the scroll position automatically "snaps" to specific points, making navigation smooth and predictable. This is especially useful for carousels, slideshows, image galleries, onboarding screens, and full-page scroll layouts.

  • scroll-snap-type: Sets the axis (x, y, block, inline) and the snapping behavior (mandatory or proximity) on the container.
  • scroll-snap-align: Defines how each child aligns to the snap point (start, center, end).
  • scroll-snap-stop: (Optional) Controls whether scrolling can skip snap points (normal or always).
  • scroll-padding & scroll-margin: Fine-tune the snap area and offset for more precise control.

Basic Syntax

/* Container */
.scroll-container {
  scroll-snap-type: x mandatory;
  overflow-x: auto;
  display: flex;
}

/* Children */
.snap-item {
  scroll-snap-align: center;
}

Modern Scroll Snap Image Gallery

.scroll-gallery-container {
  overflow-x: auto;
  scroll-snap-type: x mandatory;
  padding-bottom: 1rem;
}
.gallery-snap-track {
  display: flex;
  gap: 1.2rem;
}
.gallery-snap-item {
  flex: 0 0 70vw;
  max-width: 32rem;
  scroll-snap-align: center;
  border-radius: 0.8rem;
  overflow: hidden;
  box-shadow: 0 0.0625rem 0.5rem #0001;
  background: #fff;
}
.gallery-snap-item img {
  width: 100%;
  height: 18rem;
  object-fit: cover;
  display: block;
}
