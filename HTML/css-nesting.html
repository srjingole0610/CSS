CSS Nesting

Write cleaner, more organized, and more maintainable CSS using native nesting!

What is CSS Nesting?

CSS Nesting is a modern feature that allows you to nest selectors inside one another, similar to preprocessors like Sass, but now natively supported in CSS. This makes your stylesheets more readable and helps you logically group related rules.

  • Cleaner Syntax: Group related styles together, reducing repetition.
  • Native Support: No build tools or preprocessors neededâ€”just pure CSS!
  • Scoped Styles: Keep component or section styles together for easier maintenance.

Basic Syntax

nav {
  background: #f0f4fa;
  a {
    color: #4f8cff;
    text-decoration: none;
    &:hover {
      color: #2a4d8f;
      text-decoration: underline;
    }
  }
}

Browser Support: Native CSS nesting is supported in all modern browsers (Chrome, Edge, Safari, Firefox). Always check for the latest compatibility.
