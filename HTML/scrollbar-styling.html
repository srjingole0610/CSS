CSS Scrollbar Styling

Create beautiful, branded, and accessible scrollbars with modern CSS!

What is Scrollbar Styling?

CSS Scrollbar Styling lets you customize the look and feel of scrollbars in your web applications. You can change their color, width, border radius, and more, making them match your site's design and improving user experience.

  • WebKit Browsers: Use pseudo-elements like ::-webkit-scrollbar, ::-webkit-scrollbar-thumb, and ::-webkit-scrollbar-track.
  • Standard Properties: Use scrollbar-color and scrollbar-width for Firefox and modern browsers.
  • Accessibility: Ensure good contrast and visible focus for usability.

Syntax

/* WebKit Browsers (Chrome, Edge, Safari) */
.selector::-webkit-scrollbar {
  width: 1rem;
}
.selector::-webkit-scrollbar-thumb {
  background: #4f8cff;
  border-radius: 0.5rem;
}

/* Firefox & Modern Browsers */
.selector {
  scrollbar-color: #4f8cff #f0f4fa;
  scrollbar-width: thin;
}

.thin-scrollbar {

  scrollbar-width: thin;

.thin-scrollbar::-webkit-scrollbar {
  width: 0.4rem;
}

  background: #2a4d8f;
  border-radius: 0.3rem;
