CSS display Property

Theory & Explanation

The display property in CSS is the foundation of layout and rendering. It controls how elements are visually formatted, how they participate in the document flow, and how they interact with other elements. Mastery of display is essential for building any web layout, from simple pages to complex applications.

  • block: Starts on a new line, stretches to fill the container's width, and can have width, height, margin, and padding. Block elements stack vertically.
    Examples: <div>, <h1>, <p>, <ul>
  • inline: Flows within the current line, only as wide as its content. Width/height/margin-top/margin-bottom are ignored.
    Examples: <span>, <a>, <strong>
  • inline-block: Behaves like inline (flows in line), but you can set width, height, margin, and padding. Useful for buttons, badges, and custom controls.
  • none: The element is completely removed from the layout and accessibility tree. It is as if it does not exist in the DOM.

Other values: flex (for flexible box layouts), grid (for grid layouts), table (for table-like layouts), and more. These advanced values enable powerful, responsive designs.

How display Affects Layout

  • Block formatting context: Block elements create a new line and stack vertically. They can contain other block or inline elements.
  • Inline formatting context: Inline elements flow horizontally, wrapping as needed. They cannot have width/height set.
  • Inline-block: Inline placement, but block-like sizing and box model.
  • None: Element is not rendered and does not take up space.

Accessibility & Best Practices

  • Use semantic HTML elements for structure, then use display to adjust layout as needed.
  • Be careful with display: noneâ€”it hides content from all users, including screen readers.
  • For interactive UI, prefer inline-block or flex for custom controls and layouts.
  • Test your layout with different content lengths and screen sizes.

Code Examples

/* Block-level */
.block-demo .demo-box {
  display: block;
}

/* Inline-level */
.inline-demo .demo-box {
  display: inline;
}

/* Inline-block */
.inline-block-demo .demo-box {
  display: inline-block;
}

/* None */
.none-demo .demo-box {
  display: none;
}
