AOS (Animate On Scroll) Plugin

Bring your web pages to life with scroll-triggered animations using the AOS library and CSS!

What is AOS?

AOS (Animate On Scroll) is a popular open-source library that allows you to animate elements as you scroll down, using simple HTML attributes and CSS. Itâ€™s widely used for modern, interactive, and visually appealing web experiences.

  • Easy to use: Add data-aos attributes to any element.
  • Customizable: Choose from dozens of animation types, durations, delays, and more.
  • CSS-powered: Animations are handled via CSS transitions and transforms.
  • Responsive & performant: Works well on all modern browsers and devices.

Basic Usage

<!-- 1. Include AOS CSS & JS -->
<link rel="stylesheet" href="aos.css">
<script src="aos.js"></script>

<!-- 2. Add data-aos attribute -->
<div data-aos="fade-up">I will fade up on scroll!</div>

<!-- 3. Initialize AOS in JS -->
<script>AOS.init();</script>
